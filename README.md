> Key-Value structures: KV, Slice, Cache...

## Feature

- **Key-Value** base storage structure.
- **Non-blocking** to write.

### KV([kv.go](./kv.go))
- Value is **single** instance.

### Slice([slice.go](./slice.go))
- Value is go **slice** structure.

### Cache([cache.go](./cache.go))
- Value is **single** instance.
- Support **expire** time.
